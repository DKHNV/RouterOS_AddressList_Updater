# RouterOS_AddressList_Updater

Универсальный скрипт для роутеров MikroTik, предназначенный для обновления address list из внешних источников.

## О проекте

Данный репозиторий содержит код скрипта для RouterOS, который позволяет получать данные из внешнего источника и использовать их для автоматического обновления address list роутера.  
В моём случае в качестве источника используются GitHub Raw-файлы, которые обновляются отдельно.

## Назначение

Скрипт может применяться для:

- автоматического обновления address list на MikroTik
- централизованного использования внешних списков адресов
- упрощения сопровождения сетевых правил и маршрутизации
- снижения объёма ручной работы при обновлении списков

## Примечание

Материалы репозитория опубликованы исключительно в образовательных целях.  
Проект может использоваться для изучения принципов работы RouterOS, автоматизации сетевых задач и управления address list на роутерах MikroTik.
